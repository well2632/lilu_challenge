### Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/well2632/lilu_challenge.git
```

Entre no diretório do projeto

```bash
  cd lilu_challenge
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
